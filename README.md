
## WinlatorXR Keyboard

An on-screen keyboard for WinlatorXR to use within containers

Using the WinlatorXR InputAPI to insert input commands at a top level from the Android side, rather than within Wine via SendKeys (which may not work with many games and apps)

Thanks to erkanmaras for the original open source on-screen keyboard control project this is based on
